Who Am I?

Hi I'm Saad Ahmed. I started from getting inspired by the technological revolution which is being brought about by Artificial Intelligence (AI). I did a lot of work related to computer vision. My Final Year Project (FYP) was entitled as; Automatic Attendance System using Machine Learning.

Web development caught my interest when I realized that my FYP would get much better if I add an online/ local server based attendance record into it. That was the beginning when I started working on it. It took just four months in which the idea of a simple attendance record got groomed into a sophisticated attendance portal. The attendance portal displays the teachers’ and the students’ attendance along with the facility of viewing and downloading the pdf attendance lists.

From the day when my FYP was successfully completed, I got busy in learning web development as much as I can. I started from the basic i.e. HTML and CSS and then continued learning many other essential skills.
